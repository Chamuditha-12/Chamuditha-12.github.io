Project Description:
This project is a responsive static personal portfolio website developed using HTML5 and CSS3.
The purpose of this website is to present personal information, skills, projects, and contact details in a structured and visually appealing manner.

Theme Selected:
Personal Portfolio Website

---

Technologies Used:

* HTML5 (Semantic Elements)
* CSS3 (External Stylesheet)
* Responsive Design using Media Queries

---

Website Structure:

1. index.html       - Home page
2. about.html       - About Me page
3. projects.html    - Projects page
4. contact.html     - Contact page (Includes form)

CSS Folder:

* css/style.css     - External stylesheet used across all pages

Images Folder:

* Contains all images used in the website

---

Features Implemented:

✔ Proper HTML5 document structure
✔ Use of semantic elements (header, nav, section, footer)
✔ Navigation menu linking all pages
✔ Responsive design for mobile, tablet, and desktop
✔ Use of text, lists, images, tables, and forms
✔ Consistent styling across all pages
